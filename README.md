# Buscador de Filmes 🎬

Buscador simples feito em **React + Vite** que consulta uma API e exibe resultados na tela.

## ✅ Demo
Acesse: https://buscador-filmes-navy.vercel.app

## ✨ Funcionalidades
- Buscar filmes/séries por nome
- Exibir lista de resultados
- Estado de **loading** durante a busca
- Tratamento de **erro** quando a busca falha

## 🛠 Tecnologias
- React
- Vite
- JavaScript
- Fetch API

## ▶️ Como rodar localmente
```bash
# clonar o repositório
git clone https://github.com/NoelleAlmeida/buscador-filmes.git

# entrar na pasta
cd buscador-filmes

# instalar dependências
npm install

# rodar o projeto
npm run dev
